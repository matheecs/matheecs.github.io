### 命名约定

| 类型前缀 |                    用途 |
| :------- | ----------------------: |
| mj       |        **核心数据结构** |
| mjt      |            **主要类型** |
| mjf      |        **回调函数类型** |
| mjv      |      可视化相关数据结构 |
| mjr      | OpenGL 渲染相关数据结构 |
| mjui     |         UI 相关数据结构 |


| 函数前缀 |                用途 |
| :------- | ------------------: |
| mj_      |            核心函数 |
| mju_     |            工具函数 |
| mjv_     |      可视化相关函数 |
| mjr_     | OpenGL 渲染相关函数 |
| mjui_    |         UI 相关函数 |
| mjcb_    |    全局回调函数指针 |
